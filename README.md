A **Streamlit** web app to predict if a customer is likely to churn or stay based on their account information.  
---

## Features
- Predicts customer churn probability in real-time.  
- ML model with **recall of 91%** ✅ (good at detecting customers likely to leave).  
- Accuracy of **63%** ⚖️.  
- Simple UI with sliders and dropdowns for easy input.  
- Developed by **Syed Haseeb Shah 👨‍💻**.

---
## User Inputs
- Tenure (Months) 🗓️  
- Monthly Charges 💵  
- Contract Type 📄 (Month-to-month, One year, Two year)  
- Tech Support 🛠️ (Yes/No)  
- Internet Service 🌐 (DSL, Fiber optic, No)  
- Payment Method 💳 (Credit card, Electronic check, Mailed check)
---

## Usage
1. Clone the repo:  
git clone https://github.com/codingsheep17/customer-churn-prediction-streamlit.git
Install dependencies:
pip install -r requirements.txt
Run the app:
streamlit run app.py
